## 🧪 HOW THE FEATURES WORK TOGETHER

### User Flow - Login with Email Suggestions
```
1. User opens login page
2. Focuses on email field
3. Dropdown appears with 8 previous emails
4. User types letters to filter
5. User clicks suggestion
6. Email auto-fills
7. User enters password
8. Clicks Sign In
9. Email is saved to history file for next time
```

### User Flow - Download Appointment PDF
```
1. User goes to "My Booked Appointments"
2. Selects an appointment from table
3. QR code appears automatically (150x150)
4. User clicks "📄 Download PDF" button
5. File dialog opens
6. Defaults to Downloads folder
7. User clicks Save
8. PDF saved to: C:\Users\{username}\Downloads\INV-{appointmentId}.pdf
9. Success alert shows file location
10. QR code updates with link to downloaded PDF
```

### User Flow - Share via QR Code
```
1. User (or friend) has appointment selected
2. QR code visible
3. Using any QR scanner app on phone
4. Scans the QR code
5. Phone receives: http://localhost/pinkshield/storage/INV-{appointmentId}.pdf
6. Browser opens PDF directly
7. User can view, download, print, or email
```
